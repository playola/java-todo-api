# Java TODO API

Simple TODO API to `get`, `post`, and `delete` tasks to do. The project was bootstrapped using Spring Initializr [1]

## Start here:
```
git clone https://github.com/playola/java-todo-api.git

// Open IDE

// File > Import ... > Gradle > Existing Gradle Project
```
